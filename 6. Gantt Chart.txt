📅 6. GANTT CHART (FULL DETAIL — REPORT + TIMELINE + VIVA READY)

---

🧠 A. INTRODUCTION (FOR REPORT)

A Gantt Chart is a project management tool used to plan, schedule, and track tasks over a specific timeline. It helps in organizing activities, assigning responsibilities, and ensuring timely completion of the project.

For SahayakAI, a 1-week development timeline was followed, focusing on rapid MVP development, integration, testing, and deployment.

---

📌 B. PROJECT TIMELINE OVERVIEW

Total Duration: 7 Days
Development Type: MVP (Minimum Viable Product)
Approach: Sequential + Parallel Execution

---

📊 C. GANTT CHART TABLE (MAIN CONTENT)

👉 Use this table in your report:

| Day   | Task                                    | Description                                           | Owner              |
| ----- | --------------------------------------- | ----------------------------------------------------- | ------------------ |
| Day 1 | Problem Finalization & Dataset Creation | Define problem, collect and structure scheme data     | Data Engineer      |
| Day 2 | Backend Development                     | Build API and eligibility logic                       | Backend Developer  |
| Day 3 | Frontend Development                    | Design UI (form + layout)                             | Frontend Developer |
| Day 4 | Integration                             | Connect frontend with backend APIs                    | Backend Developer  |

| Day 6 | Testing & Debugging                     | Test system, fix bugs, validate results               | Tester             |
| Day 7 | Deployment & Demo Prep                  | Deploy app and rehearse demo                          | All Members        |
